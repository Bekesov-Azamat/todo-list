# To-Do List

A containerized To-Do List application built with a Laravel REST API and a Nuxt SPA frontend.

## Planned stack

- Laravel 13
- PHP 8.4
- Laravel Sanctum
- SQLite
- Nuxt 4
- Vue 3
- TypeScript
- Pinia
- Docker Compose

> The project is currently being bootstrapped. Complete installation,
> architecture, API, testing, and usage documentation will be added before delivery.
